## Scope (Summary)
This document provides an overview of the multi-part test suite for testing interoperability and conformance of XAdES signatures. It gives a general description of the set of test suites and lists the features of each part. For information only; normative requirements are in other parts (Parts 2–5).

## Testing Conformance and Interoperability (Overview of Parts)

### 4.1 Part 2: Test suites for testing interoperability of XAdES baseline signatures
- Specified in **ETSI TS 119 134-2 [i.5]** against **ETSI EN 319 132-1 [i.1]**.
- Test suites reflect four levels of XAdES baseline signatures: B-B, B-T, B-LT, B-LTA.
- Includes:
  - Interoperability testing for each level.
  - Augmentation testing from B-T to B-LTA.
  - Testing of signatures whose validation per [i.4] would **not** result in TOTAL_PASSED (including cases for B-B, B-T, B-LTA).

### 4.2 Part 3: Test suites for testing interoperability of extended XAdES signatures
- Specified in **ETSI TS 119 134-3 [i.6]** against **ETSI EN 319 132-2 [i.2]**.
- Test suites reflect levels of Extended XAdES signatures: XAdES-E-BES, -E-EPES, -E-T, -E-C, -E-X, -E-X-Long, -E-X-L, -E-A.
- Includes:
  - Interoperability testing for each type.
  - Testing of signatures whose validation per [i.4] would **not** result in TOTAL_PASSED (cases for -E-BES, -E-EPES, -E-T, -E-A).
  - Augmentation testing to levels -E-C, -E-X, -E-XL, -E-A.

### 4.3 Part 4: Testing Conformance for XAdES baseline signatures
- Specified in **ETSI TS 119 134-4 [i.7]** for building conformance testing tools against **ETSI EN 319 132-1 [i.1]**.
- Classifies all requirements of [i.1] into:
  1) Requirements specific to XAdES baseline signatures.
  2) Requirements common to both baseline (EN 319 132-1) and extended (EN 319 132-2) signatures.

### 4.4 Part 5: Testing Conformance for extended XAdES signatures
- Specified in **ETSI TS 119 134-5 [i.8]** for building conformance testing tools against **ETSI EN 319 132-2 [i.2]**.
- Classifies requirements from [i.1] and [i.2] into:
  1) Requirements common to both baseline and extended signatures.
  2) Requirements specific to Extended XAdES signatures.
