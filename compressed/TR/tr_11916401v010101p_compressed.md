## Scope (Clause 1)
The present document provides an overview of the set of test suites for testing interoperability and conformance of ASiC containers. It:
- a) provides a general description of the set of test suites; and
- b) lists the features of every test suite.

The present document is for information only. Normative requirements of each test suite are in other parts of this multi-part deliverable.

## Testing Conformance and Interoperability (Clause 4)

### 4.1 Part 2: Test suites for testing interoperability of ASiC baseline containers
ETSI TS 119 164-2 [i.9] specifies the test suites for testing interoperability of ASiC baseline containers against ETSI EN 319 162-1 [i.1].

ETSI EN 319 162-1 [i.1] defines four different levels of ASiC baseline containers. The test suites include layers reflecting those levels:
- B-B level conformance
- B-T level conformance
- B-LT level conformance
- B-LTA level conformance
- Augmentation from B-T to B-LTA
- ASiC baseline containers where signature validation (per ETSI EN 319 102-1 [i.4]) would not result in TOTAL_PASSED (test cases for B-B, B-T, B-LTA levels).

### 4.2 Part 3: Test suites for testing interoperability of additional ASiC containers
ETSI TS 119 164-3 [i.10] specifies the test suites for testing interoperability of additional ASiC containers against ETSI EN 319 162-2 [i.2].

ETSI EN 319 162-2 [i.2] defines additional containers including time assertions and signatures as defined in ETSI EN 319 122-2 [i.5] and ETSI EN 319 132-2 [i.6]. Test suites for containers including signatures follow ETSI TS 119 124-3 [i.7] (CAdES extended) and ETSI TS 119 134-3 [i.8] (XAdES extended).

### 4.3 Part 4: Testing Conformance for ASiC baseline containers
ETSI TS 119 164-4 [i.11] defines requirements for building software tools for testing technical conformance of ASiC baseline containers against ETSI EN 319 162-1 [i.1]. The requirements in [i.1] are classified into two groups:
1. Requirements specific to ASiC baseline containers.
2. Requirements common to both ASiC baseline containers (EN 319 162-1) and additional ASiC containers (EN 319 162-2).

### 4.4 Part 5: Testing Conformance for additional ASiC containers
ETSI TS 119 164-5 [i.12] defines requirements for building software tools for testing technical conformance of additional ASiC containers against ETSI EN 319 162-2 [i.2]. The requirements in [i.1] and [i.2] are classified into two groups:
1. Requirements common to both ASiC baseline containers (EN 319 162-1) and additional ASiC containers (EN 319 162-2).
2. Requirements specific to Additional ASiC containers (EN 319 162-2).
