# ETSI TS 119 124-3: CAdES digital signatures - Testing Conformance and Interoperability; Part 3: Test suites for testing interoperability of extended CAdES signatures
**Source**: ETSI | **Version**: V1.1.1 | **Date**: 2016-06 | **Type**: Technical Specification  
**Original**: http://www.etsi.org/standards-search (or direct link)

## Scope (Summary)
Defines test suites to assess interoperability between implementations claiming conformance to extended CAdES signatures [2] per ETSI EN 319 122-2 levels. Includes test suites for each signature level, augmentation, and negative cases. Agnostic to PKI infrastructure.

## Definitions and Abbreviations
- **negative test case**: test case for a signature whose validation according to ETSI EN 319 102-1 would not result in TOTAL-PASSED
- Abbreviations as per ETSI TR 119 001 [i.2] apply.

## 4 Overview
Test suites are layered per extended CAdES levels [2]. Covers:
- Testing CAdES signatures: BES, EPES, T, C, X, XL, A (with ATSv2/v3)
- Testing negative CAdES signatures: BES, EPES, T, A
- Testing augmentation: to C, X, XL, A levels
