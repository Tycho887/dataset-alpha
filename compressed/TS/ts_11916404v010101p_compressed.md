# ETSI TS 119 164-4: Electronic Signatures and Infrastructures (ESI); Associated Signature Containers (ASiC) - Testing Compliance and Interoperability; Part 4: Testing Conformance of ASiC baseline containers
**Source**: ETSI | **Version**: V1.1.1 | **Date**: 2016-06 | **Type**: Normative
**Original**: [DTS/ESI-0019164-4](http://www.etsi.org/standards-search)

## Scope (Summary)
Defines the set of checks for testing conformance of ASiC containers against ASiC baseline containers as specified in ETSI EN 319 162-1 [1]. Does not specify checks for technical validity (e.g., cryptographic validity). Checks are constrained to elements specified by ASiC [1]. Test assertions include tests on correctness of structure of all elements specified by ASiC [1].

## 4 ASiC baseline container conformity test specification overview
Three conformance testing layers are defined:
1. **Container Structure (CS)**: presence of required files with specific names.
2. **Syntactical Conformance (SC)**: correctness of file syntax/properties.
3. **Signature Conformance (STV)**: conformance of CAdES or XAdES signatures contained in the container.

All layer tests **shall** be performed to complete testing. No tests for technical validity of cryptographic material.

Each TA includes: unique identifier, Normative Source reference, Target, Prerequisite(s), Predicate (normative condition), Prescription Level (mandatory/recommended/optional), Tag.
