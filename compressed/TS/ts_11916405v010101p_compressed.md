# ETSI TS 119 164-5: Electronic Signatures and Infrastructures (ESI); Associated Signature Containers (ASiC) - Testing Compliance and Interoperability; Part 5: Testing Conformance of additional ASiC containers
**Source**: ETSI TC ESI | **Version**: V1.1.1 | **Date**: June 2016 | **Type**: Technical Specification (Normative)  
**Original**: [ETSI portal](http://www.etsi.org/standards-search)

## Scope (Summary)
Defines conformance test assertions for ASiC containers conforming to building blocks in EN 319 162-1 and additional containers in EN 319 162-2. Complements TS 119 164-4 (baseline container testing) by adding tests for extended CAdES/XAdES signatures and additional container types. Test assertions follow OASIS Test Assertions Guidelines.

## Definitions and Abbreviations
- **ACS**: Additional Container Structure (first conformance layer)
- **ACSC**: Additional Container Syntactical Conformance (second conformance layer)
- **ACST**: Additional Container Signature or Timestamp Token conformance (third conformance layer)
- **TA**: Test Assertion (as per OASIS [i.1])
- **TAL**: Test Assertion List
- **TC**: Test Case
- **PKI**: Public Key Infrastructure
- **URI**: Uniform Resource Identifier
- For general terms, refer to ETSI TR 119 001 [i.3]; in case of conflict, ETSI TR 119 001 prevails.

## 4 ASiC Additional Container Conformity Test Specification Overview
The document defines conformance tests for:
1. ASiC additional containers expanding baseline containers (using extended CAdES [i.4] or extended XAdES [i.5]) per EN 319 162-1 clause 4.
2. ASiC additional containers as specified in EN 319 162-2.
Tests extend TS 119 164-4 [5] with extended CAdES/XAdES tests and define new tests for additional container types.
Each TA contains: unique ID, normative source, target, prerequisite, predicate, prescription level (mandatory/recommended/permitted), and tag.
