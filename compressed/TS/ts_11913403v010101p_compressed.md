# ETSI TS 119 134-3 V1.1.1: Electronic Signatures and Infrastructures (ESI); XAdES digital signatures - Testing Conformance and Interoperability; Part 3: Test suites for testing interoperability of extended XAdES signatures
**Source**: ETSI | **Version**: V1.1.1 | **Date**: 2016-06 | **Type**: Normative (Technical Specification)
**Original**: http://www.etsi.org/standards-search

## Scope (Summary)
Defines test suites to assess interoperability between implementations claiming conformance to extended XAdES signatures as specified in ETSI EN 319 132-2 [2]. Covers all levels defined in that standard, augmentation of extended XAdES signatures, and negative test cases. Agnostic of PKI infrastructure.

- [2] ETSI EN 319 132-2: "XAdES digital signatures; Part 2: extended XAdES signatures"
- [3] ETSI TS 119 134-2: "XAdES digital signatures - Testing Conformance and Interoperability; Part 2: Test suites for testing Interoperability of XAdES baseline signatures"

## Definitions and Abbreviations
- **negative test case**: test case either for a signature that is not an extended XAdES signature, or for a signature whose validation according to ETSI EN 319 102-1 would not result in TOTAL_PASSED
- **Abbreviations**: CA (Certification Authority), CRL (Certificate Revocation List), OCSP (Online Certificate Status Provider), TSA (Time-Stamping Authority)
- **XML Namespace Prefixes**: `ds` = http://www.w3.org/2000/09/xmldsig#; `xades` = http://uri.etsi.org/01903/v1.3.2#; `xadesv141` = http://uri.etsi.org/01903/v1.4.1#
- **Modal verbs**: shall, shall not, should, may, need not, etc. are interpreted as per ETSI Drafting Rules clause 3.2. "must" not allowed except in direct citation.

## 4 Overview
Describes overall approach to specify test suites for extended XAdES signatures per ETSI EN 319 132-2 [2]. Eight levels defined. Test suites include interoperability tests, negative tests, and augmentation tests. Certain extended signatures are also baseline signatures; therefore test cases from ETSI TS 119 134-2 [3] are reused where applicable.

## 7 Test suites with negative test cases

### 7.1 Introduction
Defines negative test cases for extended XAdES signatures. Two types: (1) test cases leading to valid signatures that are not XAdES signatures; (2) test cases leading to extended XAdES signatures whose validation would not result in TOTAL_PASSED.

### 7.2 Test cases generating non XAdES signatures
The test case XAdES/NB/4 defined in ETSI TS 119 134-2 [3] clause 10.2 generates a XML signature that is not a XAdES signature.
