oidc
The most comprehensive OpenIdConnect package for dart.
This is a Mono repo of multiple packages:
oidc_core
:
The main dart package, it implements the basic openid/oauth specification, except the platform specific parts (like redirection)
oidc
: builds on top of
oidc_core
, and adds platform-specific parts, this also contains the
UserManager
class.
oidc_default_store
: the default
OidcStore
store implementation.
oidc_cli
: a provider-agnostic CLI for authenticating against an OIDC provider.
